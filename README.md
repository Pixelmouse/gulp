# Gulp

## Instalar

1) Asegurate de tener instalado el cliente de gulp.

``` sh
npm install gulp-cli -g
```

2) Descarga `gulpfile.js` y `package.json`.

3) Instala dependencias.

``` sh
npm i
```

## Uso

### Compilar

``` sh
gulp
```
